<footer>...</footer>

Document Object Model (DOM) Explanation

  • <html> - Root element containing entire document
  • <head> - Metadata, title, and CSS links
  • <body> - Visible content of the webpage
  • Marquee - Scrolling announcement bar
  • Navigation - Menu linking all pages
  • Header/Hero - Main banners across the site
  • Sections - Content areas varying per page
  • Footer - Copyright, contact info, and structural links
